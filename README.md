## Description
Compares two configuration files and shows a difference.
## Installation
npm package:
```sh
$ npm install -g sergey20x25-gd
```
## Launch
To start type **gendiff** in the console:
```sh
$ gendiff
```
## Help
For help type **gendiff -h** in the console:
```sh
$ gendiff -h
```
